# C++ implementation

This is a C++ implementation for BounceFilter and related solutions.

---

## Project Structure

```
BounceFilter_Cpp/
├── main.cpp
├── header/
│   ├── MurmurHash3.h
│   ├── Sketch.h
│   ├── CountMin.h
│   ├── BounceFilter.h
│   └── LogLogFilter.h
├── MurmurHash3.cpp
├── CountMin.cpp
├── BounceFilter.cpp
└── LogLogFilter.cpp
```

## Usage

### Compilation

```bash
$ mkdir build
$ cd build
$ cmake ..
$ make
```

### Execution

```bash
$ ./BounceFilter_Cpp
```

## Requirements

- CMake 3.26 or above
- Compiler with support for C++17 standard
